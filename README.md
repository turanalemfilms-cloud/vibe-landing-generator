# Vibe Landing Bot

Telegram bot for generating landing page leads.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the bot:
```bash
node bot.js
```

## Environment Variables

Create a `.env` file (or set in your hosting provider):

```
TELEGRAM_TOKEN=your_token_here
OWNER_ID=your_telegram_id
WEBAPP_URL=https://your-site-url.com
```

## Deploy

This bot is ready to be deployed on Railway, Render, or any Node.js hosting.
